# 车道线二分类感知 — 简短实验报告

---

**模型**: ResNet18 (10.6M) | **数据**: TuSimple 6408 帧 | **框架**: PyTorch 2.x + ONNX Runtime  
**训练策略**: Focal Loss + WeightedSampler + AMP | **设备**: GPU (CUDA)

---

## 1. 数据集

**原始来源**: TuSimple Lane Detection Dataset，共 6408 个标注帧 (1280×720)。

**构造方法**: 每张图以图像中心 (x=640) 为基准，从底部向上裁剪 640×360 的车辆前方 ROI。标签判定：在 y=690 处，若图像中心位于左右自车道边界之间 → `lane_in (1)`，否则 → `lane_out (0)`。

| 划分 | 样本数 | lane_in | lane_out | lane_in% |
|------|--------|---------|----------|----------|
| Train | 5128 | 3812 | 1316 | 74.3% |
| Val | 640 | 473 | 167 | 73.9% |
| Test | 640 | 477 | 163 | 74.5% |

## 2. 模型与训练配置

**主干网络**: ResNet18 (10.6M 参数, ImageNet 预训练)。

**训练策略**:
- 损失函数: Focal Loss (α=0.25, γ=2.0)
- 采样: WeightedRandomSampler (~50:50/batch)
- 优化器: AdamW (lr=1e-3, wd=1e-4)
- 调度器: CosineAnnealingWarmRestarts (T_0=10)
- 精度: AMP 混合精度 (CUDA)
- Batch Size: 128 | Epochs: 18 (早停) | 训练耗时: ~8 分钟 (GPU)

## 3. 测试集结果

### v1 (MobileNetV2, 500样本) vs v2 (ResNet18, 6408样本) 对比

| 指标 | v1 | v2 | 提升 |
|------|-----|-----|------|
| Accuracy | 80.00% | **91.56%** | +11.56 pp |
| Precision | 81.97% | **94.53%** | +12.56 pp |
| Recall | 92.59% | **94.13%** | +1.54 pp |
| F1 Score | 86.96% | **94.33%** | +7.37 pp |
| lane_out Recall | 47.6% | **84.0%** | +36.4 pp |

## 4. 加分项完成情况

1. **TensorBoard**: `train.py` 内置 SummaryWriter，训练曲线 (loss/accuracy/LR) 保存至 `outputs/runs/`，可用 `tensorboard --logdir outputs/runs` 查看。
2. **ONNX 导出与推理**: `export_onnx.py` 导出 → `lane_binary.onnx` (43MB)，`infer_onnx.py` 使用 ONNX Runtime 执行推理，精度误差 < 1e-7。
3. **数据增强**: 训练集使用 RandomHorizontalFlip(p=0.3) + ColorJitter(brightness/contrast/saturation/hue) + RandomRotation(±5°)，不参与增强的验证/测试集仅归一化。

## 5. 结论

基于 ResNet18 + Focal Loss 的二分类方案在 TuSimple 全量数据集上取得了 Accuracy=91.56%、F1=94.33%、lane_out 召回率=84.0% 的性能。Focal Loss + WeightedRandomSampler 的组合将 lane_out 召回率从 47.6% 提升至 84.0%（+36.4 pp），是 v2 的核心改进。模型支持 ONNX 导出与跨平台推理部署，指标达到工程应用水平。
